# Emotion Detection App (NLP + Machine Learning)

This is a simple **Streamlit-based web application** that detects emotions from text using a trained Machine Learning model with TF-IDF features.

---

## 🚀 Features

- Detects emotions from text input
- Uses TF-IDF vectorization for text processing
- Machine Learning model for classification
- Displays emotion with emojis 😊😢😡😨❤️😲
- Simple and interactive Streamlit UI

---

## 🧠 Emotions Detected

- Joy 😊  
- Sadness 😢  
- Anger 😡  
- Fear 😨  
- Love ❤️  
- Surprise 😲  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit 🌐  
- Scikit-learn 🤖  
- NLP (TF-IDF Vectorizer)  
- Pickle (Model saving/loading)
